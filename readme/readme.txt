How to operate VOS:
1. To enter a voice command please say VOS/Hey VOS,Has to be said before any other commands(Moving glasses indicate vos is ready to listen)
2. To set VOS in sleep or hibernation just say stop/sleep/cancel.
3. To start application, simply speak the name of the application or say launch + the name of the desired application
4. To control system settings such as volume, brightness, speak command name + up/down/on/off
5. To control applications, make sure the application is open and then enter your desired command, such as copy/paste/new window/new tab
